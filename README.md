## PROJECTS

<td width="50%" valign="top">

### [AUTOCRAWL](https://github.com/Y0EL/autocrawl)
`AUTONOMOUS INTELLIGENCE CRAWLER`

LangGraph state machine that discovers security and defense expo vendors across the open web. 50 parallel workers. Multi tier search pipeline spanning Wikipedia REST, DuckDuckGo, and OpenSERP across Google, Bing, Yandex, and Baidu. PDF OCR via Ollama vision. Vue 3 tactical dashboard with 2.5D world map and live graph canvas. 12 Docker services. Zero cloud dependency by default.

**929 vendors indexed in a single overnight run. 12x the original estimate.**

`LangGraph` `LangChain` `Ollama` `Crawl4AI` `FastAPI` `PostgreSQL` `Redis` `Vue 3` `MapLibre`

</td>
<td width="50%" valign="top">

### [ATLAS](https://github.com/Y0EL/ATLAS)
`LOCAL AI VOICE PIPELINE`

End to end local voice AI for the UIX intelligence system. Whisper handles speech to text with CUDA 12.x GPU acceleration including Blackwell RTX 50xx support. LLM layer routes to Groq, OpenRouter, or Ollama. edge-TTS converts output to speech. WebSocket live mode, persistent memory layer, model benchmark suite. Built to run entirely on device.

`Whisper ASR` `Ollama` `Groq` `edge-TTS` `FastAPI` `CUDA 12.x`

</td>
</tr>
<tr>
<td width="50%" valign="top">

### [GEOCEK](https://github.com/Y0EL/geocek)
`OSINT GEOLOCATION ENGINE`

Visual signal to coordinate system. Extracts license plate prefix, road classification, lane count, landmark mentions, and infrastructure layout from images then intersects constraints to narrow candidates. Confidence scored output in GeoJSON and interactive Folium maps. 100 percent free APIs only.

`Python` `OSM Nominatim` `Overpass API` `GPT-4o` `LangChain` `Folium` `Mapbox`

</td>
<td width="50%" valign="top">

### [SENTINEL](https://github.com/Y0EL/Sentinel)
`ANTIFRAUD HEURISTICS ENGINE`

Behavioral signal analysis for fraud detection. Heuristic scoring across transaction patterns and interaction data. Detects inauthentic activity through pattern variance analysis. Built from first principles understanding of how platforms distinguish organic from synthetic behavior.

`Python` `Signal Analysis` `Behavioral Patterns` `Heuristics`
